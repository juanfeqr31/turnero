# Sukha Frontend (Next.js)

Minimal frontend scaffold for the Sukha project.

Setup:

1. Install dependencies

```bash
cd frontend
npm install
```

2. Create `.env` based on `.env.example` and run dev

```bash
npm run dev
```

Environment variables:

- `BACKEND_API_URL` - base URL of the FastAPI backend (e.g. `http://localhost:8000`)
